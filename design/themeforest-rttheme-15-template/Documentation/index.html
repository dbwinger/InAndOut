Markup Markup customization and highlights

This theme is a fixed layout with two columns. All of the information within the main content area is nested within a div with an id of "container". The sidebar's (left column) content is within a div with an id of "sidebar".

Change Structure

If you find that your new style is not overriding, it is most likely because of a specificity problem. Scroll down in your CSS file and make sure that there isn't a similar style that has more weight.

#wrap #primaryContent a {
   color: #someColor;
}

How to customize template color?

I have included some psd files into the PSD Files folder. In order to get customized left-sidebar you can use these files. For example;

  • If you're using the defaul skin (grey one) open /skin-default/skin-default.psd
  • Change the color of the file as you wish
  • Save as skin-default.png in /images/ folder and replace with the old one
  • Once you customized the curvy images you need to edit css also. Open /css/style.css and change background value of the class as your customized image. (see example code #1)
  • If you're working another skin, you need to customize it's own color. You can find specific CSS codes for the other skins at the bottom of the file. Example for the blue skin (see example code #2) 

/*  example code #1*/
 
/* sidebar background color */
  .sidebar_content{
    background:#343434;
    padding:0 30px;
 }

/* example code #2*/
 
body.blue-skin .sidebar_content{background:#0c3348; }

How to customize Body Fonts?

I have used Google Fonts API

http://www.google.com/webfonts

for the template. Open the style.css and include your fonts as I did. 

/* ----------------------------------------------------

	GOOGLE FONTS

------------------------------------------------------- */
	/* Oswald */
	@import url(http://fonts.googleapis.com/css?family=Oswald);

	/* Droid Sans */
	@import url(http://fonts.googleapis.com/css?family=Droid+Sans:400,700);

	/* Droid Serif */
	@import url(http://fonts.googleapis.com/css?family=Droid+Serif:400,400italic,700,700italic);

	/* body */
	body {
		font-family: 'Droid Sans', Arial, Helvetica, sans-serif;

		position:relative;
		background:#E5E5E5;
		color:#666;
		padding:0 0 0 1px;
	}

Customize Cycle Slider

If you want to change the cycle slider's speed transition and it's effect, open up the file scripts.js and change the following code: 

  /* CYCLE SLIDER */
  var rttheme_slider_timeout=7000; //miliseconds 7000 = 7 seconds
  var rttheme_slider_effect="fade";

Customize Nivo Slider

If you want to change the slider's speed transition, open up the file scripts.js and change the following code: 

  /* NIVO SLIDER */
  var rttheme_nivo_slider_timeout=7000; //miliseconds 7000 = 7 seconds

Twitter Feeds

Find these lines in the scripts.js and change the value with your twitter username: 

  /* TWITTER */ 
  var twitter_user_name = 'envatowebdesign'; //your twitter username  
  var tweet_count = 1; //number of tweets to show

Flickr Feeds

Find these lines in the scripts.js and change the value with your flickr username:
